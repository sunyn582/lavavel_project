Laravel Online Store

🛒 Giới thiệu

Laravel Online Store là một hệ thống thương mại điện tử hoàn chỉnh, xây dựng bằng Laravel, hỗ trợ quản lý sản phẩm, danh mục, giỏ hàng, đặt hàng, đánh giá, phân quyền người dùng, giao diện hiện đại và dễ mở rộng cho nhiều mục đích kinh doanh.

🛠️ Công nghệ sử dụng

  • Laravel 12: Framework PHP mạnh mẽ cho phát triển web.
  • Laravel Breeze: Định danh & xác thực người dùng (đăng ký, đăng nhập, xác thực email, quên mật khẩu).
  • CRUD đầy đủ cho Product: Quản lý sản phẩm (tạo, xem, sửa, xóa) với validation, upload ảnh, phân quyền.
  • TailwindCSS: Thiết kế giao diện hiện đại, responsive.
  • MySQL: Lưu trữ dữ liệu.
  • Seeder & Migration: Tạo dữ liệu mẫu và quản lý cấu trúc database.
  • Bảo mật: CSRF, XSS, xác thực, phân quyền, session, cookies, validation, chống SQL Injection, policy, middleware.
  • RESTful API: Hỗ trợ thao tác CRUD qua API cho Product.
  • Session & Cookies: Quản lý đăng nhập, bảo vệ thông tin người dùng.
  • Eloquent ORM: Truy vấn dữ liệu an toàn, tránh SQL Injection.
  • Aiven Cloud Database: Sử dụng Eloquent để migrate và đồng bộ dữ liệu các Object trực tiếp lên database cloud (Aiven).

🌟 Danh sách tính năng chi tiết

  • Quản lý sản phẩm: Thêm, sửa, xóa sản phẩm, upload ảnh, phân trang, tìm kiếm, lọc danh mục, xem chi tiết.
  • Quản lý danh mục: CRUD danh mục, gán sản phẩm, lọc, hỗ trợ slug SEO.
  • Giỏ hàng & đặt hàng: Thêm/xóa/cập nhật sản phẩm trong giỏ, đặt hàng, quản lý đơn hàng.
  • Đánh giá sản phẩm: Người dùng đánh giá, nhận xét, hiển thị điểm trung bình.
  • Quản trị viên: Đăng nhập admin, quản lý toàn bộ dữ liệu, giao diện riêng, bảo vệ middleware.
  • Người dùng: Đăng ký, đăng nhập, cập nhật thông tin, xem lịch sử đơn hàng, đổi/quên mật khẩu.
  • Giao diện: Responsive với TailwindCSS, navbar, phân trang, thông báo, xác nhận thao tác.
  • Seed dữ liệu mẫu: Seeder cho sản phẩm, danh mục, user.
  • Bảo mật & hiệu năng: CSRF, phân quyền, validate, cache danh mục/sản phẩm.

⚡ Hướng dẫn cài đặt & sử dụng

  1. Clone dự án: 
    git clone https://github.com/your-username/laravel-online-store.git
cd laravel-online-store
  2. Cài đặt Composer & NPM: 
    composer install
npm install
  3. Tạo file môi trường: 
    cp .env.example .env
php artisan key:generate
  4. Cấu hình database rồi migrate & seed: 
    php artisan migrate --seed
  5. Build frontend: 
    npm run build
  6. Khởi động server: 
    php artisan serve
    Truy cập: http://localhost:8000
